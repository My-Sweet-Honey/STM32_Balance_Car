# 基于 STM32F103C8T6 的智能平衡车

## **项目功能**
1. 配备陀螺仪和加速度计传感器，实时监测车体姿态，确保平衡稳定
2. 能够使用蓝牙进行遥控，实现前进、后退、转弯等基本操作
3. OLED 显示屏实时显示传感器数据和车体状态
4. 采用 PID 控制算法，优化平衡性能和响应速度

## **硬件选型**
1. **主控芯片**：STM32F103C8T6
2. **传感器**：MPU6050
3. **显示屏**：OLED 显示屏
4. **电机驱动**：TB6612FNG
5. **电机**：JGA25-370 直流减速电机
6. **蓝牙模块**：HC-05
7. **电源**：2节 18650 锂电池（7.4V）
8. **降压模块**：MP1584EN（将电压降至 5V）

## **软件设计**
1. **开发环境**：Keil uVision 5
2. **编程语言**：C 语言
3. **开发方式**：固件库/标准外设库 开发
4. **主要程序**：见代码
